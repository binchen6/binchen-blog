# 尘墨 · 国风古代科技风设计系统

## 设计哲学

"以古为镜，以技为骨"——将中国传统美学中的水墨意境、青铜纹饰、天文意象与现代科技元素相融合，打造一套兼具东方韵味与科技质感的设计语言。

### 核心理念
- **宣纸为底**：以传统宣纸的温润质感为视觉基底
- **墨色为骨**：以墨色的浓淡变化构建层次
- **青铜为饰**：以青铜器的纹饰与色泽点缀细节
- **星图为灵**：以天文星图、罗盘的秩序感构建科技气质
- **朱砂为印**：以印章的朱砂红作为点睛之笔

## 纹理与装饰

### 核心纹理

1. **宣纸纹理** (paper-texture)
   - 使用 SVG noise filter 生成微妙噪点
   - 透明度 3-5%，叠加于背景之上
   - 营造宣纸的纤维质感

2. **星图网格** (star-grid)
   - 以 30px 为间隔的淡线网格
   - 交叉点添加微小星点（2px 圆点）
   - 透明度 8-12%，营造天文仪器感

3. **青铜回纹** (bronze-pattern)
   - 用于边框、分隔线装饰
   - 可旋转的 SVG 几何图案

4. **水墨晕染** (ink-wash)
   - 径向渐变圆形，边缘模糊
   - 用于背景装饰、区块过渡

### 装饰元素

1. **罗盘指针**
   - 缓慢旋转（60-120s/圈）
   - 用于 Hero 区域装饰

2. **齿轮装饰**
   - 交错旋转的齿轮 SVG
   - 用于科技相关区块

3. **云纹边框**
   - 中国传统云纹图案
   - 用于卡片边缘、分隔线

4. **二十八宿星点**
   - 散布的微小发光点
   - 用于深色背景装饰

## 组件风格

### 按钮

| 类型 | 样式 |
|------|------|
| **主按钮** | 墨黑背景，宣纸白文字，悬停时反色 + 青铜边框 |
| **次要按钮** | 透明背景，墨黑边框，悬停时填充墨黑 |
| **科技按钮** | 暗青背景，青铜金文字，带微光效果 |
| **印章按钮** | 朱砂红边框，朱砂红文字，旋转 -2deg |

### 卡片

- 背景：宣纸暖白渐变
- 边框：1px 淡墨（透明度 6%）
- 悬停：上浮 4px + 阴影扩散
- 角落：可选青铜回纹装饰

### 输入框

- 背景：宣纸白
- 边框：1px 淡墨，聚焦时变为暗青 + 微光
- 文字：等宽字体（用于代码/数据场景）

### 导航栏

- 背景：宣纸白 + 80% 透明度 + 毛玻璃效果
- 底部边框：1px 淡墨
- Logo：篆书风格 "尘" 字

### 核心动效

| 动效 | 时长 | 缓动 | 用途 |
|------|------|------|------|
| **墨水晕染** | 1.5s | ease-out | 区块入场 |
| **缓慢旋转** | 120s | linear | 罗盘装饰 |
| **上浮悬停** | 0.4s | ease | 卡片交互 |
| **下划线展开** | 0.4s | ease | 链接悬停 |
| **印章弹跳** | 0.3s | ease | 印章悬停 |
| **粒子飘散** | 3s | ease-out | 背景装饰 |

### 页面过渡

- 使用 Framer Motion 的 `AnimatePresence`
- 淡入 + 微上移（y: 20 → 0）
- 时长 0.6-0.8s
